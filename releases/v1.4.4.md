# JellyPanel v1.4.4 — Per-Server Live TV Connection Fix

This release restores the missing backend implementation used by **Live TV & EPG → Sync selected server**.

## Fixes

- Creates or updates the selected Jellyfin server's GalaxyTV M3U tuner.
- Uses the secured server-specific M3U feed URL.
- Creates, reuses, updates, and removes GalaxyTV-managed XMLTV providers independently per server.
- Safely recreates stale Jellyfin tuner or listing-provider IDs.
- Persists tuner/provider state in `jellyfin_live_tv_connections`.
- Starts Jellyfin's Refresh Guide task after a successful connection.
- Retains v1.4.3 M3U rendering and v1.4.2 fleet-resilience fixes.

Database schema remains **19**.

## Runtime test

After upgrading, select the Jellyfin server and click **Sync selected server**. Confirm the status becomes **Connected**, then verify the GalaxyTV tuner and XMLTV provider in the Jellyfin dashboard.
